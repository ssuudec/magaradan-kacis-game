## Amaç

**Kırmızı butona basıp çıkış kapısından geç** — ama her bölümde farklı bir zorluk var:

## Kontroller

- **Hareket:** WASD / Ok tuşları
- **Zıpla:** Space / W / ↑
- **Sıfırla:** R
- **Fare:** Bölüm 7, 8, 10'da sürükleme ve tıklama
